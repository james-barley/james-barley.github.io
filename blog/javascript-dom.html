Javascript & the Dom

  • An analogy of HTML and CSS

    HTML holds all the content like the structure of a house and the shape and CSS is the interior designer it changes the colours the fonts and every design aspect on the page.

  • Control Flow

    The control flow is the order in which we do things if I want to put on my shoe I need my sock on first and my shoe before thinking about doing up my laces. Loops in control flow are processes where the instructions need to repeat until a certain condition is met

  • The DOM

    The DOM (Document object model) is an interface to interact with web pages. We use the Dom to add, remove and change content on our web pages.

  • Arrays & objects

    They are types of different groupings, arrays is for when the variables are all the same type such as different ages or names and objects is when you want a grouping of lot of different types of variables.

  • Functions

    A function is a set of statements that performs a task. It takes into account variables and pushes out another variable.
